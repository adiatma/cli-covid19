# cli-covid19

Simple CLI (Command Line Interface) to get covid19 reporting by country

## Install

```
cargo install cli-covid19
```

## Usage

```bash
> cli-covid19 id
+-------------+----------------------------+
| Last Update | 2020-08-10 02:35:19 +00:00 |
+-------------+----------------------------+
| Confirmed   | 125396                     |
+-------------+----------------------------+
| Recovered   | 80952                      |
+-------------+----------------------------+
| Deaths      | 5723                       |
+-------------+----------------------------+
```
